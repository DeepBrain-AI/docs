# Docs to video

Convert your documents into videos using Deepbrain AI's advanced technology. To check the progress of the video generation, see [progress](../progress) endpoint.

<br />

## 1. API endpoint

```
POST https://app.aistudios.com/api/odin/v3/automation/docs-to-video
```

<br />

## 2. Request parameter

| key                    | desc                                                                                                                                                    | type                                             | required | default    |
| :--------------------- | :------------------------------------------------------------------------------------------------------------------------------------------------------ | :----------------------------------------------- | :------- | :--------- |
| files                  | File uri retrieved from the [file-upload](./upload-files) api                                                                                                             | String[]                                         | true     | -          |
| options                | The configuration to be used for the video generation                                                                                                   | Json                                             | false    | {}         |
| options.goal           | The purpose of the video generation                                                                                                                     | 'auto' \| 'business' \| 'youtube' \| 'education' | false    | 'business' |
| options.duration       | The expected duration of the generated video                                                                                                            | 'auto' \| Number                                 | false    | -          |
| options.speed          | The video playback speed compared to the original speed                                                                                                 | 'auto' \| Number                                 | false    | -          |
| options.language       | The language used in the video. Note that the language code follows the [ISO 639-1](https://www.loc.gov/standards/iso639-2/php/code_list.php) standard. | 'auto' \| String                                 | false    | -          |
| options.media          | The source of the image assets used in the video generation (only valid if options.fileBackground is false)                                             | 'auto' \| 'search' \| 'free' \| 'generative'     | false    | -          |
| options.style          | The style of the generative assets (only valid if options.media is 'generative')                                                                        | 'auto' \| 'business' \| 'youtube' \| 'education' | false    | -          |
| options.fileBackground | Whether to use the file you have provided in the background of the video                                                                                | Boolean                                          | false    | true       |
| options.orientation    | The screen orientation of the generated video                                                                                                           | 'web' \| 'mobile'                                | false    | 'web'      |
| options.model          | The id of the AI model to be used for the video generation                                                                                              | String                                           | false    | -          |

:::info File Types

If any of the parameters in option object is not provided, default values will be used.

:::

<br />

## 3. Response parameters

| key       | desc                            | type   |
| :-------- | :------------------------------ | :----- |
| projectId | ID of the created video project | String |
