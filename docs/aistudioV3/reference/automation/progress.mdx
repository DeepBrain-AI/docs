# Progress

Check the progress of the automated AI video generation. To see how to export the generated video and check out the download URL, see the [project export](../export-project) endpoint.

<br />

## **1. API Endpoint**

```
GET https://app.aistudios.com/api/odin/v3/automation/progress?projectId=${projectId}
```

<br />

## **2. Response Parameters**

The API responds with a JSON object containing the progress details of the video generation.

| Key        | Description                                       | Type    |
|------------|---------------------------------------------------|---------|
| state | Current state of the automation process          | String  |
| progress | Completion percentage of the video generation | Number  |

<br />
